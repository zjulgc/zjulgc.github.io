On the Impact of Multi-language Development in Machine Learning Frameworks

  • RQ1: 机器学习框架中多语言开发的流行程度如何？
    • ML 框架和传统系统在多语言开发和多语言拉取请求的比例方面具有可比性。
  • RQ2:多语言开发对机器学习框架中的拉取请求接受率有什么影响？
    • ML 框架中的多语言 PR 的接受率低于单语言 PR。
  • RQ3:多语言开发对机器学习框架中接受拉取请求的周期有什么影响？
    • ML 框架中的多语言 PR 比单语言 PR 需要更长的时间被接受，并且 ML 框架接受/拒绝多 PR 比传统系统需要更长的时间。
  • RQ4:在机器学习框架中，多语言拉取请求是否比单语言拉取请求更容易出错？
    • ML 框架的单语言 PR 比传统系统更容易出错。

AOBTM: Adaptive Online Biterm Topic Modeling for Version Sensitive Short-texts Analysis

本文主要是对一个应用不同版本的评论进行主题的建模，来知道每个版本的更新

Main Idea

  • 应用的评论是短文本，所以传统的topic模型不能适用，其也不能大量连续时间片的主题，本文的自适应在线双项主题模型（AOBTM）自适应地对短文本中的主题进行建模，AOBTM 缓解了短文本中的稀疏性问题，并考虑了统计数据以获得最佳数量的先前时间片。 同时提出了并行算法来自动确定主题推理阶段应考虑的最佳主题数量和最佳先前版本数量。对应用程序评论和真实世界短文本数据集的自动评估证实 AOBTM 可以找到更连贯的主题并优于最先进的基线。
