JavaScript-0

Intro.

JavaScript是脚本语言，是可插入 HTML 页面的编程代码

位置

HTML 中的脚本必须位于 <script> 与 <script> 标签之间。

脚本可被放置在 HTML 页面的 <body> 和 <head> 部分中。

  • 通常把函数放入 <head> 部分中，或者放在页面底部。这样就可以把它们安置到同一处位置，不会干扰页面的内容。
  • 脚本的数量是没有限制的

输入输出流

  • 使用 window.alert() 弹出警告框。
  • 使用 document.write() 方法将内容写到 HTML 文档中。
  • 使用 innerHTML 写入到 HTML 元素。
  • 使用 console.log() 写入到浏览器的控制台。

调用另一文件写好的js脚本

  • 外部脚本不能包含<script> 标签

操作HTML元素

如需从 JavaScript 访问某个 HTML 元素，可以使用 document.getElementById(id) 方法。使用 “id” 属性来标识 HTML 元素，并 innerHTML 来获取或插入元素内容

变量

这里需要注意的一点是：JS中的变量可以先使用后声明，这个也被叫做变量的提升

但是初始化并不会提升

  • 实例 2 的 y 输出了 undefined，这是因为变量声明 (var y) 提升了，但是初始化(y = 7) 并不会提升，所以 y 变量是一个未定义的变量

严格模式

为了避免变量提升，在JS中我们可以使用"use strict"即严格模式，在这个模式中，我们不能使用没有定义的变量

其他限制

  • 不允许使用未声明的变量

  • 不允许删除变量或对象

  • 不允许删除函数

  • 不允许使用八进制

  • 不允许对只读属性赋值

  • 不允许对一个使用getter方法读取的属性进行赋值

  • 不允许删除一个不允许删除的属性

  • 变量名不能使用 “eval” 字符串

  • 变量名不能使用 “arguments” 字符串

  • 不允许使用以下这种语句

    • "use strict";
with (Math){x = cos(2)}; // 报错

  • 函数（Function）字面量 定义一个函数

    • function myFunction(a, b) { return a * b;}

  • 一条语句中可以对多个变量进行赋值，使用逗号进行分隔计科

    • var lastname="Doe", age=30, job="carpenter";

运算符

其他的基本与常见语言的运算符是一致的，只是js多了一对比较运算符

  • ===表示绝对等于，这代表两个变量的值与类型均与另一个变量一致
  • !==表示不绝对等于，值和类型有一个不相等，或两个都不相等

null与undefined

  • undefined：是所有没有赋值变量的默认值，自动赋值
  • null：主动释放一个变量引用的对象，表示一个变量不再指向任何对象地址
